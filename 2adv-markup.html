    Links

    Links use the a tag and display inline. They must have an href attribute and can use target to open in a new tab

    Links have different psuedo-class states, if they're hovered or have been visited, for example.

    Images

    Images use the img tag, which is self-closing, and they mignt be inside a figure tag. Images use the src = attribute to point the the source, which could be anywhere. Images display by default.
